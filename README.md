# 🎮 Romeo Mini Games

4款经典HTML小游戏合集 — 零依赖、纯前端、即开即玩！

## 🕹️ 游戏列表

| 游戏 | 类型 | 核心算法 |
|------|------|----------|
| [俄罗斯方块](tetris.html) | 方块消除 | 网格碰撞检测 + 行消除判定 |
| [五子棋](gomoku.html) | 双人/AI对战 | AI评分算法 + 胜负判断 |
| [扫雷](minesweeper.html) | 经典扫雷 | Flood Fill 展开算法 |
| [坦克大战](tanks.html) | 射击闯关 | BFS寻路 + 炮弹预判 |

## ✨ 功能特性

- **零依赖** — 每个游戏都是独立HTML文件，无需安装任何框架或库
- **Canvas渲染** — 使用原生 Canvas 2D API 实现流畅画面
- **中英文双语** — 内置 i18n 系统，一键切换语言（`data-lang-key` 驱动）
- **深色/浅色主题** — CSS变量 + localStorage 持久化用户偏好
- **响应式设计** — 适配桌面和移动端浏览器

## 🚀 本地运行

双击任意 `.html` 文件即可在浏览器中运行。

## 🛠️ 技术栈

- HTML5 Canvas 2D
- CSS Variables（主题系统）
- Vanilla JavaScript（ES6+）
- localStorage（用户偏好持久化）
- 无外部依赖，零构建步骤
